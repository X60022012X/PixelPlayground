Tutorial

1. Objective:

Your task is to uncover all the clear squares and mark all the mines on the grid, without clicking on any mines. Each square you uncover will either reveal a number indicating the number of adjacent mines, or it will be blank if there are no mines nearby.

2. Controls:

      For PC:

    • Click: Uncover a square.
    • Hold "space" and click: Place a flag on a cell that contains a mine.

      For Mobile:

    • Select "Uncover cell" and click: Uncover a square.
    • Select "Mark mine" and click: Place a flag on a cell that contains a mine.

3. Gameplay:

  • Start by uncovering any square on the grid.
  • The number revealed on a square indicates how many mines are adjacent to it.
  • Use the numbers as clues to deduce the locations of mines.
  • If you uncover a square with a mine, the game ends.
  • If you find a square that must contain a mine, use a flag to mark it as a mine.
  • You can remove a flag by clicking te´he square agian.

4. Strategy Tips:

  • Use logic and deduction to identify safe squares.
  • Pay attention to patterns and use process of elimination to narrow down the locations of mines.
  • Go to setting to change, for example: minefield size and the amount of mines.
  • The score is based off and the amount of mines, minefield size and time remaining.

5. Winning:

  • Clear all the safe squares and mark all the mines on the grid, without detonating any mines to win the game.
  • Once you've uncovered all safe squares and marked all the mines on the grid, the game will automatically end.

6. Losing:

  • If you uncover a square with a mine, the game ends immediately, and you lose.
  • Take your time, strategize, and think carefully before making each move to avoid uncovering any mines accidentally.
